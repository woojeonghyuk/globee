# 🌍 Globee

외국인 유학생(선생님)과 초등학생(학생)을 연결하여  
문화 교류 기반 수업을 제공하는 플랫폼

---

## 🧩 서비스 개요

Globee는  
유학생이 자신의 문화와 언어를 활용해 수업을 개설하고,  
초등학생은 다양한 국가의 문화를 체험할 수 있는 플랫폼입니다.

---

## 👤 사용자 유형

### 1. 학생 (초등학생)
- 수업 탐색 및 신청
- 다양한 국가 문화 체험
- 외국인과 교류 경험

### 2. 선생님 (유학생)
- 수업 개설
- 자신의 문화/언어 기반 수업 진행
- 수익 창출

---

## 🔐 핵심 기능 (MVP 기준)

### ✅ 인증
- 회원가입 / 로그인
- 사용자 타입 선택 (학생 / 선생님)

### ✅ 학생 기능
- 수업 목록 조회
- 수업 신청

### ✅ 선생님 기능
- 수업 개설
- 수업 관리

---

## 🧠 서비스 흐름

1. 사용자는 로그인 시 "학생 / 선생님" 선택
2. 선생님 → 수업 생성
3. 학생 → 수업 목록 조회 후 신청
4. 매칭 후 수업 진행
